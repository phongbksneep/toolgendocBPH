# HƯỚNG DẪN SỬ DỤNG BPHH-DocGen (Windows)

## 1) Cài đặt

- Tải file `BPHH-DocGen-Setup-1.0.6.exe` từ trang release.
- Chạy file cài đặt và hoàn tất wizard.

Nếu không muốn cài đặt, có thể dùng bản portable:
- `BPHH-DocGen-1.0.6-x64.zip` → giải nén và chạy `BPHH-DocGen.exe`.

---

## 2) Chuẩn bị dữ liệu

Trong app, chọn **📤 Xuất / Nhập**:
- **📄 File JSON mẫu** hoặc
- **📊 File Excel mẫu (.xlsx)**

Điền dữ liệu theo label gợi ý, sau đó lưu file.

---

## 3) Nhập dữ liệu vào app

- Nếu dùng JSON: bấm **📂 Mở JSON**.
- Nếu dùng Excel: **📤 Xuất / Nhập → 📊 Import từ Excel**.

---

## 4) Tạo hồ sơ DOCX

- Kiểm tra danh sách job bên trái (bật/tắt theo nhu cầu).
- Bấm **▶ Generate DOCX**.
- Chọn thư mục output.
- App tạo xong sẽ hiện popup kết quả.

---

## 5) Quản lý template

- Bấm **📐 Templates** để:
  - xem template,
  - sửa template,
  - tạo template mới,
  - import template `.docx`.

---

## 6) Lưu ý quan trọng

- App tự động lưu phiên làm việc (autosave).
- Label trên form là label nghiệp vụ để người dùng điền cho dự án mới.
- Nếu thấy dữ liệu hiển thị bất thường, khởi động lại app và mở lại file JSON/Excel.

---

## 7) Phiên bản hiện tại

- App version: **1.0.6**
- Kênh phát hành: Windows (khách hàng mục tiêu dùng Win)
